# 暗黑模式主题系统 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: 使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务执行本计划。步骤用 checkbox（`- [ ]`）语法追踪。

**Goal:** 为「摆烂随笔」小程序增加亮/暗/跟随系统三态主题系统，切换即时全局生效并持久化，纯前端实现。

**Architecture:** 把 9 个设计色 token 从硬编码改为 `var(--c-*)` CSS 变量，在 `tokens.scss` 里用 `.theme-light`/`.theme-dark` 两套变量定义配色；因小程序 JS 无法改 page 属性，用一个 `PageLayout` 容器组件在每个页面根节点挂 `theme-{resolved}` class，让 CSS 变量在整页子树生效；主题态由 zustand store（`store/theme.ts`）驱动、纯函数 `resolveTheme` 解析、`Taro.setStorageSync` 持久化、`app.tsx` 里 `onThemeChange` 跟随系统。

**Tech Stack:** Taro 4.2.0、React、TypeScript、zustand、weapp-tailwindcss 3.7、bun、vitest。

## Global Constraints

- 语言：所有新增注释与文案用中文（zh-CN），保留必要技术术语。
- 测试命令：`bun run test`（vitest run）。测试文件放 `src/**/__tests__/*.test.ts`，`environment: node`、`globals: true`（无需 import vitest 全局）。
- 类型检查：`bunx tsc --noEmit`。
- 小程序编译：`bun run build:weapp`（`taro build --type weapp`，产物在 `dist/`）。
- 小程序无 DOM：JS 不能 `setAttribute('data-theme')`，必须用 PageLayout 容器挂 class（方案 A），禁止依赖 page 属性注入。
- PageLayout 必须包整个页面内容（含 ScrollView 的兄弟节点，如 profile 的 SettingsDrawer），否则 CSS 变量作用域漏。
- 不依赖 `display:contents`（小程序支持不稳）；PageLayout 用普通 View + `height:100%` + `flex column`，保证内部 ScrollView（`height:100vh` / `min-h-screen`）撑满，不破坏滚动。
- 暗色配色（Design Doc §4，逐值 verbatim）：
  - `.theme-light`: bg `#FAF6F0` / card `#FFFFFF` / card-soft `#FEFCF9` / peach `#F0A868` / taro `#E4A9BE` / haze `#A6C0CE` / ink `#4A413A` / ink-sub `#8A7F76` / heart `#EF8A7F` / shadow `rgba(74,65,58,0.08)`
  - `.theme-dark`: bg `#1A1613` / card `#26201B` / card-soft `#2E2721` / peach `#F0A868` / taro `#D98FAA` / haze `#7E9AA8` / ink `#E8E2D9` / ink-sub `#A89E93` / heart `#E87B70` / shadow `rgba(0,0,0,0.3)`
  - peach 亮暗两套均保持 `#F0A868`。
- 主题 store 默认 `mode: 'system'`；`resolveTheme(mode, sys) = mode === 'system' ? sys : mode`。
- `onThemeChange` 在 app 级只注册一次，避免泄漏/重复。
- 富文本用户内容内联色不强制跟主题（非目标）；不做切换过渡动画（即时切换）。
- 每个任务独立 commit，遵循 conventional commits（feat/refactor/test/chore）。

## Task 4: 主题 store（store/theme.ts）

zustand store：内存态 + 持久化，复用 `store/auth.ts` 的 `Taro.setStorageSync` 模式。

**Files:**
- Create: `src/store/theme.ts`

**Interfaces:**
- Consumes: `resolveTheme`、`Mode`、`Resolved`（Task 3）。
- Produces: `useThemeStore`（zustand），state 形如：
  - `mode: Mode`（默认 `'system'`）
  - `resolved: Resolved`（默认 `'light'`）
  - `setMode: (m: Mode) => void` —— 存 mode、持久化 `theme-mode`、用当前系统主题重算 resolved
  - `applySystem: (sysTheme: Resolved) => void` —— system 模式下更新 resolved，非 system 忽略
  - `init: (sysTheme: Resolved) => void` —— 启动读持久化 mode 并算 resolved（供 app.tsx 调）

- [ ] **Step 1: 写 store**

创建 `src/store/theme.ts`：

```ts
import { create } from 'zustand'
import Taro from '@tarojs/taro'
import { resolveTheme, type Mode, type Resolved } from '../utils/theme'

const MODE_KEY = 'theme-mode'

interface ThemeState {
  mode: Mode
  resolved: Resolved
  /** 用户手选模式：存内存 + 持久化 + 用系统主题重算 resolved */
  setMode: (m: Mode) => void
  /** onThemeChange 回调：system 模式下跟随系统更新 resolved */
  applySystem: (sysTheme: Resolved) => void
  /** 启动初始化：读持久化 mode（默认 system），用系统主题算 resolved */
  init: (sysTheme: Resolved) => void
}

export const useThemeStore = create<ThemeState>((set, get) => ({
  mode: 'system',
  resolved: 'light',

  setMode: (m) => {
    const sys = getSystemTheme()
    Taro.setStorageSync(MODE_KEY, m)
    set({ mode: m, resolved: resolveTheme(m, sys) })
  },

  applySystem: (sysTheme) => {
    if (get().mode === 'system') set({ resolved: sysTheme })
  },

  init: (sysTheme) => {
    const stored = (Taro.getStorageSync(MODE_KEY) as Mode) || 'system'
    set({ mode: stored, resolved: resolveTheme(stored, sysTheme) })
  },
}))

/** 读当前系统主题，缺省按 light */
function getSystemTheme(): Resolved {
  try {
    return (Taro.getSystemInfoSync().theme as Resolved) || 'light'
  } catch {
    return 'light'
  }
}
```

- [ ] **Step 1: 写 PageLayout**

创建 `src/components/PageLayout/index.tsx`：

```tsx
import { ReactNode } from 'react'
import { View } from '@tarojs/components'
import { useThemeStore } from '../../store/theme'

/**
 * 主题容器：根 View 挂 theme-{resolved} class，让 CSS 变量在整页子树生效。
 * 用 height:100% + flex column 透传布局，保证内部 ScrollView（100vh/min-h-screen）撑满。
 * 必须包整个页面内容（含 ScrollView 的兄弟节点），否则变量作用域漏。
 */
export default function PageLayout({ children }: { children: ReactNode }) {
  const resolved = useThemeStore((s) => s.resolved)
  return (
    <View
      className={`theme-${resolved}`}
      style={{ height: '100%', display: 'flex', flexDirection: 'column' }}
    >
      {children}
    </View>
  )
}
```

- [ ] **Step 3: app.tsx 初始化主题 + 注册 onThemeChange**

修改 `src/app.tsx`：在 `useLaunch` 内初始化主题，并在 app 级注册一次 `onThemeChange`。

```tsx
import { PropsWithChildren } from 'react'
import Taro, { useLaunch } from '@tarojs/taro'
import { useAuthStore } from './store/auth'
import { useThemeStore } from './store/theme'
import Toast from './components/Toast'
import './app.scss'

function App({ children }: PropsWithChildren) {
  useLaunch(() => {
    // 启动时从本地存储恢复登录态（token 未失效时自动登录）
    useAuthStore.getState().restore()

    // 主题初始化：读持久化 mode + 系统主题算 resolved
    let sysTheme: 'light' | 'dark' = 'light'
    try {
      sysTheme = (Taro.getSystemInfoSync().theme as 'light' | 'dark') || 'light'
    } catch {
      sysTheme = 'light'
    }
    useThemeStore.getState().init(sysTheme)

    // app 级只注册一次：system 模式跟随系统深浅色实时切换
    Taro.onThemeChange(({ theme }) => {
      useThemeStore.getState().applySystem(theme as 'light' | 'dark')
    })
  })

  return (
    <>
      {children}
      <Toast />
    </>
  )
}

export default App
```

```tsx
import { View, Text } from '@tarojs/components'
import Taro from '@tarojs/taro'
import { logout } from '../../services/auth'
import { useThemeStore } from '../../store/theme'
import type { Mode } from '../../utils/theme'

export interface SettingsDrawerProps {
  open: boolean
  onClose: () => void
  /** 退出登录成功后回调（供个人中心清本地态） */
  onLoggedOut?: () => void
}

const THEME_OPTIONS: { label: string; value: Mode }[] = [
  { label: '亮', value: 'light' },
  { label: '暗', value: 'dark' },
  { label: '跟随系统', value: 'system' },
]

/** 右滑全屏设置抽屉：主题三选 / 草稿箱 / 退出登录 */
export default function SettingsDrawer({ open, onClose, onLoggedOut }: SettingsDrawerProps) {
  const mode = useThemeStore((s) => s.mode)
  const setMode = useThemeStore((s) => s.setMode)

  const goDrafts = () => {
    onClose()
    Taro.navigateTo({ url: '/pages/drafts/index' })
  }

  const onLogout = async () => {
    const { confirm } = await Taro.showModal({ title: '退出登录', content: '确定退出当前账号吗？' })
    if (!confirm) return
    logout()
    onClose()
    onLoggedOut?.()
  }

  return (
    <View
      className='fixed'
      style={{
        top: 0,
        left: 0,
        right: 0,
        bottom: 0,
        zIndex: 100,
        visibility: open ? 'visible' : 'hidden',
      }}
      catchMove={open}
    >
      {/* 遮罩 */}
      <View
        className='fixed'
        style={{
          top: 0,
          left: 0,
          right: 0,
          bottom: 0,
          background: 'rgba(0,0,0,0.3)',
          opacity: open ? 1 : 0,
          transition: 'opacity .28s ease',
        }}
        onClick={onClose}
      />
      {/* 右侧面板 */}
      <View
        className='fixed bg-bg'
        style={{
          top: 0,
          right: 0,
          bottom: 0,
          width: '80%',
          transform: open ? 'translateX(0)' : 'translateX(100%)',
          transition: 'transform .28s ease',
          zIndex: 101,
        }}
      >
        <View className='px-6 pt-16 pb-8 flex flex-col' style={{ height: '100%' }}>
          {/* 顶部标题 + 关闭 */}
          <View className='flex items-center justify-between mb-8'>
            <Text className='text-lg text-ink font-bold'>设置</Text>
            <View className='press' onClick={onClose}>
              <Text className='text-sm text-ink-sub'>关闭</Text>
            </View>
          </View>

          {/* 主题三选 */}
          <View className='bg-card rounded-card shadow-soft px-4 py-4 mb-4'>
            <Text className='text-base text-ink'>主题</Text>
            <View className='flex mt-3'>
              {THEME_OPTIONS.map((opt) => {
                const activeItem = mode === opt.value
                return (
                  <View
                    key={opt.value}
                    className={`press rounded-pill px-4 py-2 mr-3 ${activeItem ? 'bg-peach' : 'bg-card-soft'}`}
                    onClick={() => setMode(opt.value)}
                  >
                    <Text className={`text-sm ${activeItem ? 'text-card' : 'text-ink-sub'}`}>
                      {opt.label}
                    </Text>
                  </View>
                )
              })}
            </View>
          </View>

          {/* 草稿箱入口 */}
          <View
            className='press bg-card rounded-card shadow-soft px-4 py-4 mb-4'
            onClick={goDrafts}
          >
            <Text className='text-base text-ink'>草稿箱</Text>
          </View>

          {/* 退出登录（置底） */}
          <View
            className='press mt-auto py-3 flex justify-center items-center rounded-card'
            style={{ border: '1rpx solid var(--c-taro)' }}
            onClick={onLogout}
          >
            <Text className='text-sm' style={{ color: 'var(--c-taro)' }}>
              退出登录
            </Text>
          </View>
        </View>
      </View>
    </View>
  )
}
```
